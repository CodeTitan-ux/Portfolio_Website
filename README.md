# Portfolio_Website

This is a 3D Portfolio website built with React, TypeScript, and Three.js.

## Tech Stack
- **React 19**
- **Three.js** (via @react-three/fiber & @react-three/drei)
- **Framer Motion**
- **Vite**
- **Tailwind CSS 4**

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/CodeTitan-ux/Portfolio_Website.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
